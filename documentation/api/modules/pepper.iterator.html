Module pepper.iterator

Revision log iterator. This class is used for iterating over a given revision range. Use pepper.repository:iterator to construct a revision iterator.

Functions

map (callback) Calls callback for all remaining revisions.
next () Returns the next revision, or nil if there are no more revisions.
revisions () Returns a Lua iterator for the remaining revision ids.
