Tarballs

pepper is distributed as a source code package. You can grab the latest release from the list at GitHub or use these links:

  • pepper-0.3.2.tar.gz
  • pepper-0.3.2.zip

All releases are available at GitHub. The plain-text change log summarizes the changes between subsequent releases.

Please see the Documentation page for installation instructions and basic usage information.

Distribution packages

Here's a list of packages, ready to install for the respective distributions:

  • Arch Linux (AUR)
  • Gentoo (James Rowe's misc-overlay)
  • Debian Sid
  • Ubuntu Raring Ringtail

For source packages, please remember to install optional build-time dependencies for the repository backends first (e.g., subversion and mercurial on Arch Linux).

Thanks to all maintainers!

Git repository

The latest, possibly unstable source code can be obtained from pepper's Git repository at GitHub.
